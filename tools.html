Cloud - incluir no servidor

As inclusões devem ser realizadas quando no status do serviço no próprio equipamento é apresentado como: ACESSO NEGADO / PERMISSION DENIED / NÃO REGISTRADO

Usuário: flavio
Senha: 767254#_

• Inclusão

Na dúvida da informação do NS, solicite uma foto do menu Intelbras Cloud para que o mesmo seja cadastrado corretamente.
Não é possível cadastrar NS com caracteres com letras minúsculas.
Não é possível cadastrar NS com apenas números.

Após realizada a inclusão no servidor o prazo padrão para a conclusão da solicitação é de uma hora, após este período o equipamento deve ser reinicializado.
